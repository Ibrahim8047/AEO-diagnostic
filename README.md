## What is AEO?

**Answer Engine Optimization (AEO)** is the next frontier of SEO. As AI-powered search (Google AI Overview, Gemini, ChatGPT) replaces traditional blue links, brands must now optimize to be *recommended by AI* — not just ranked by algorithms.

This tool gives you a real-time **report card** on who owns the answer layer for any search query.

---

## Features

- 🔍 **Google SERP Analysis** — Fetches top 10 organic results in real-time via SerpAPI
- 🤖 **Gemini AI Response** — Queries Gemini 2.5 Flash to see which brands it recommends
- 🌟 **Google AI Overview** — Captures Google's featured AI answer box (highest-value AEO real estate)
- 📊 **AI-Powered Scoring** — Gemini acts as a judge and scores top 5 brands across 4 dimensions:
  - Google SERP Rank
  - Gemini AI Mentions
  - Sentiment
  - Visibility
- 🧠 **Strategic Intelligence** — Landscape analysis + growth opportunity for challenger brands
- ⚡ **Live Dashboard** — Beautiful dark-themed UI with KPI cards, rankings table, and score breakdown

## Getting Started

### 1. Clone the repo

```bash
git clone https://github.com/Ibrahim8047/AEO-diagnostic.git
cd AEO-diagnostic
```

### 2. Install dependencies

```bash
pip install -r requirements.txt
```

### 3. Run the app

```bash
streamlit run main.py
```

### 4. Add your API keys in the app

- **SerpAPI Key** → [serpapi.com](https://serpapi.com)
- **Gemini API Key** → [aistudio.google.com](https://aistudio.google.com)

Keys are entered in-app and never stored or shared.
